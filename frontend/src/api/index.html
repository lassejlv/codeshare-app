API Documention v1.0.0

This is the documentation for our API. You can use this API to create snippets directly from your application. No API key is required.

Rate Limits

We have a rate limit of 15 requests per minute. If you exceed this limit you will be blocked for 1 minute.

Endpoints

The base URL for all endpoints is https://api.shareyourcode.app

All endpoints are prefixed with /api/:route

Routes

The following routes are available:

  • POST - Payload: { "snippet": { "title": "Hello World", "code": "console.log('Hello World')" "language": "js" } } /snippet/new - Create a new snippet
